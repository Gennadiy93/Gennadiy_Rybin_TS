Название тестового набора: Тестирование домашней страницы веб-сайта HackerRank. В автотестах файл test_home_page.py
Цель: Проверить корректность отображаемой информации и переход на страницу регистрации аккаунта
Предположения: Веб-сайт доступен и функционирует нормально.
Ограничения: Тесты разработаны для браузера Chrome

Тестовые кейсы:
1. Отображение  HackerRank лого на домашней странице
Шаги:
1.1 Инициализация драйвера
1.2 Открытие страницы https://www.hackerrank.com
Ожидаемый результат: Открытая домашняя страница и отображение логитипа HackerRank

2. Отображение двух Sign up кнопок на домашней странице
Шаги:
2.1 Инициализация драйвера
2.2 Открытие домашней страницы https://www.hackerrank.com
Ожидаемый результат: Наличие двух Sign up кнопок на домашней странице

3. Открытие Get Started страницы после клика на одну из sign up кнопок на домашней странице
Шаги:
3.1 Инициализация драйвера
3.2 Открытие страницы https://www.hackerrank.com
3.3 Нажатие на одну из двух кнопок Sign Up
Ожидаемый результат: Открытая Get Started страница и отображение корректного заголовка страницы

4. Открытие страницы Sign up после выбора free account на странице Get Started
Шаги:
4.1 Инициализация драйвера
4.2 Открытие страницы https://www.hackerrank.com/get-started/
4.3 Нажатие на кнопку  'I’m here to practice and prepare'
4.4 Нажатие на кнопку "Create account"
Ожидаемый результат: Открытая Get Started страница и отображение корректного заголовка страницы

Название тестового набора: Тестирование регистрации free account на веб-сайте HackerRank. В автотестах файл test_sign_up_page.py
Цель: Проверить корректность работы процесса регистрации free acoount на веб-сайте.
Предположения: Веб-сайт доступен и функционирует нормально.
Ограничения: Тесты разработаны для браузера Chrome

1. Регистрация free account используя валидные данные, остановка на первом шаге анкетирования
Шаги:
1.1 Инициализация драйвера
1.2 Открытие страницы https://www.hackerrank.com/auth/signup
1.3 Генерация тестовых данных (fullname, email, password)
1.4 Заполнение формы данными (fullname, email, password)
1.5 Нажатие на чекбокс согласия с условиями предоставления услуг
1.6 Нажатие на кнопку Create Account
Ожидаемый результат: Открытая страница с анкетированием и совпадение отображаемого name с введенным на странице заполнения формы

2. Регистрация free account используя валидные данные с пропуском анкетирования
Шаги:
2.1 Инициализация драйвера
2.2 Открытие страницы https://www.hackerrank.com/auth/signup
2.3 Генерация тестовых данных (fullname, email, password)
2.4 Заполнение формы данными (fullname, email, password)
2.5 Нажатие на чекбокс согласия с условиями предоставления услуг
2.6 Нажатие на кнопку Create Account
2.7 Нажатие на один из двух ответов на первый вопрос
2.8 Нажатие на один из двух ответов на второй вопрос
2.9 Нажатие на кнопку "Skip" для пропуска анкетирования
2.10 Нажатие на иконку крестика модального окна
Ожидаемый результат: Закрытие модального окна и наличие Profile Menu. Пользователь успешно зарегистрирован

3. Регистрация free account используя невалидный email
Шаги:
3.1 Инициализация драйвера
3.2 Открытие страницы https://www.hackerrank.com/auth/signup
3.3 Генерация тестовых данных (fullname, invalid email, password)
3.4 Заполнение формы данными (fullname, invalid email, password)
Ожидаемый результат: Отображаются введенные данные и наличие сообщения о некорректном email

4. Регистрация free account используя валидные данные с последующем выходом из системы
Шаги:
4.1 Инициализация драйвера
4.2 Открытие страницы https://www.hackerrank.com/auth/signup
4.3 Генерация тестовых данных (fullname, email, password)
4.4 Заполнение формы данными (fullname, email, password)
4.5 Нажатие на чекбокс согласия с условиями предоставления услуг
4.6 Нажатие на кнопку Create Account
4.7 Нажатие на один из двух ответов на первый вопрос
4.8 Нажатие на один из двух ответов на второй вопрос
4.9 Нажатие на кнопку "Skip" для пропуска анкетирования
4.10 Нажатие на иконку крестика модального окна
4.11 Нажатие на кнопку Profile Menu
4.12 Нажатие на кнопку Sign out
Ожидаемый результат: Выход из системы, отсутствие Profile menu, наличие кнопок Log In и Sign Up

Замечания:  Данный веб-сайт позволяет регистрироваться с тестовыми емейлами. Описаны автотесты только для регистрации free account. Веб-сайт также позволяет осуществлять регистрацию для компаний с 14-дневным бесплатным периодом.
